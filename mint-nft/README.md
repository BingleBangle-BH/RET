# RET

Quick start

```
cd mint-nft
yarn install
npm install ts-node --save-dev
yarn add ts-node
yarn test
```

Start middleware(API)
Examples are given in the middleware for access each end point. Use GET request.
```
pip install
cd endpoints
python3 middleware.py
```

create a new secret file and add in the following
- 'destSecret.json'
- 'ownerSecret.json'
